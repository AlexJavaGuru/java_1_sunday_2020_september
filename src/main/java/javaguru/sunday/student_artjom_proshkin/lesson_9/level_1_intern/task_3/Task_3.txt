Возможно ли в Java объявить класс и сделать его приватным,
не видимым для всего другого кода? Подумайте почему?
Ответ на этот вопрос закомитьте в текстовом файле.

Приватный класс можно объявить только внутри другого класса (inner class).
Родительский (супер-класс) приватным быть не может.
Приватность класса делает бесполезным его использование (наследование,вызовы методов, конструкторов) в коде.
